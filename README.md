## 📖 项目简介 | Introduction

欢迎来到 **上海电力大学（SUEP）专升本考试复习资料仓库**。

本项目旨在收集、整理并分享上海电力大学“专升本”选拔考试的相关复习资源。希望通过开源的方式，打破信息差，帮助每一位备考的同学更高效地复习，顺利上岸！🚀

> **注意**：本项目为非官方仓库，所有资料仅供个人学习交流使用，严禁用于商业用途。

## 🛠 使用指南 | Usage

1. **浏览/下载**：点击上方文件夹即可查看对应科目的资料。如果需要下载单个文件，直接点击文件后点击 `Download` 或 `Raw`。

2. **批量下载**：

   - 点击仓库右上角的 `Code` -> `Download ZIP` 下载全部资料。

   - 或者使用 Git 命令克隆到本地：

     ```
     git clone https://github.com/yuexuan521/SUEP-Exam.git
     ```

## 🤝 如何贡献 | Contribution

众拾柴火焰高！如果你手头有本仓库缺失的资料（如最新的真题、优秀的复习笔记等），欢迎贡献！

### 方式一：Pull Request (推荐)

1. **Fork** 本仓库到你的 GitHub 账号。
2. 将你的资料上传到对应的文件夹（如果没有对应文件夹，可以新建）。
3. 提交更改 (Commit) 并 Push 到你的仓库。
4. 向本仓库发起 **Pull Request (PR)**。

### 方式二：Issues

如果你不熟悉 Git 操作，也可以在 [Issues](https://www.google.com/search?q=https://github.com/yuexuan521/SUEP-Exam/issues) 中发布网盘链接，我们会定期整理上传。
