1、数据结构与算法和C++代码大题难吗？

- 代码题不会那么难，题型都是固定的

2、数据结构与算法代码大题用什么语言？

- 算法设计题是写伪代码就行了，参考408考研数据结构与算法大题

3、会不会超纲？

- 数据结构与算法：排序得全学

4、数据结构与算法和C++代码大题要写注释吗？

- 最好写注释

5、题库写几遍？

- 20遍有没有

6、2025年考生成绩排名收集

- 93 91  89  87  85 83 81 80 79 78  76  75  74

7、考试问题

- 九点开考，提前十五分钟进考场
- 考试前一天可以去看考场
- 提前订酒店（因为旁边是建桥，有超多考生）
- 考试试卷跟模拟卷很像，钉在一起，没有答题卡写卷子上，全部是手写（包括代码大题，没有上机）

10、有错的答案
